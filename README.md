# Анализ рынка подержанных автомобилей в Preset

Проект представляет собой BI-дашборд для анализа рынка подержанных автомобилей на основе датасета с объявлениями о продаже машин.

Дашборд построен в Preset.io и позволяет изучить структуру рынка, сравнить бренды по цене, посмотреть зависимость цены от пробега и подобрать конкретные автомобили под заданные условия.

## Цель проекта

Цель проекта — показать, как с помощью BI-инструмента можно решить практическую задачу выбора подержанного автомобиля.

Пользовательский сценарий:

Пользователь хочет купить подержанный автомобиль, например с бюджетом до 400 000.
С помощью фильтров он выбирает подходящий диапазон цены, бренд, год выпуска, пробег и тип топлива.
После этого дашборд показывает, какие автомобили подходят под заданные условия, какие бренды чаще встречаются в выборке и как цена связана с пробегом.

## Используемые данные

Источник данных: Kaggle, датасет по подержанным автомобилям Quikr.

В проекте используются два файла:

* `data/raw/quikr_car.csv` — исходный датасет;
* `data/clean/used_cars_clean.csv` — очищенный датасет, подготовленный для загрузки в Preset.

Исходный датасет содержит следующие поля:

* `name` — название автомобиля;
* `company` — бренд автомобиля;
* `year` — год выпуска;
* `Price` — цена автомобиля;
* `kms_driven` — пробег;
* `fuel_type` — тип топлива.

## Подготовка данных

Перед загрузкой в Preset данные были очищены и преобразованы в Jupyter Notebook:

* `scripts/01_prepare_used_cars_dataset.ipynb`

В ходе подготовки данных были выполнены следующие шаги:

* поле `Price` приведено к числовому формату;
* значение `Ask For Price` обработано как пропуск;
* поле `kms_driven` очищено от текста `kms` и приведено к числовому формату;
* поле `year` приведено к числовому формату;
* добавлено поле `car_age` — возраст автомобиля;
* добавлено поле `price_segment` — ценовой сегмент;
* добавлено поле `kms_segment` — сегмент пробега;
* обработаны пропуски в поле `fuel_type`;
* удалены некорректные значения в поле `company`, которые не являлись брендами автомобилей.

## Дашборд

Дашборд создан в Preset.io.

Ссылка на дашборд: https://a5dabecc.us1a.app.preset.io/superset/dashboard/p/YAmjv6jvxQ2/

Демонстрация работы дашборда:

![Демонстрация дашборда](screenshots/dashboard_demo.gif)


В дашборде реализованы следующие блоки:

### KPI-показатели

* количество объявлений;
* средняя цена автомобиля;
* средний пробег;
* средний возраст автомобиля.

### Графики

* средняя цена по брендам;
* топ брендов по количеству объявлений;
* зависимость цены от пробега;
* таблица со списком автомобилей.

### Фильтры

В дашборд добавлены фильтры, позволяющие пользователю сузить выборку:

* цена;
* бренд;
* год выпуска;
* пробег;
* тип топлива.

## Пользовательский сценарий

Пример пользовательского пути:

1. Пользователь открывает дашборд.
2. Смотрит общую картину рынка: количество объявлений, среднюю цену, средний пробег и средний возраст автомобилей.
3. Устанавливает фильтр по бюджету, например до 400 000.
4. Дополнительно фильтрует автомобили по бренду, году выпуска, пробегу и типу топлива.
5. Анализирует, какие бренды чаще всего попадают в выбранные условия.
6. Смотрит график зависимости цены от пробега.
7. В таблице выбирает конкретные автомобили, подходящие под заданные параметры.

Такой сценарий помогает пользователю быстро понять, какие автомобили доступны в рамках его бюджета и какие варианты выглядят наиболее подходящими.
